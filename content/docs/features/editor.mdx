The editor provides a rich interface for reviewing and refining generated content after initial generation.

## Features

- **AI Refinement** — give instructions like "make it shorter" or "add more examples" and the AI rewrites the content
- **SEO Scoring** — real-time SEO analysis with actionable improvement suggestions
- **Version History** — track changes across refinement iterations
- **Auto-Save** — content saves automatically as you edit
- **Export** — download as Markdown, HTML, or PDF

## AI Refinement

1. Open any generation in the editor
2. Type a refinement instruction (e.g., "Add a call to action at the end")
3. The AI regenerates the content incorporating your instruction
4. The original version is preserved in version history

## SEO Scorer

The built-in SEO scorer analyzes content and provides:

- Readability score
- Keyword density analysis
- Heading structure check
- Content length assessment
- AI-powered improvement suggestions

Click **Get AI Suggestions** for specific recommendations, then **Apply Suggestions** to let the AI implement them automatically.

## Repurpose from Editor

From any generation in the editor, click **Repurpose** to transform the content into social media formats using the [repurpose](/docs/features/repurpose) feature.
