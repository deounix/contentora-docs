Brand voices let users define their company's tone, audience, and personality so every piece of generated content stays on-brand.

## Creating a Brand Voice

1. Go to **Brand Voices** in the sidebar
2. Click **Create Brand Voice**
3. Fill in the profile:

| Field | Description |
| --- | --- |
| Name | Profile name (e.g., "Formal Corporate") |
| Company Name | Your brand or company |
| Industry | Your industry (tech, healthcare, etc.) |
| Target Audience | Who the content is for |
| Tone Keywords | Tags like "professional", "friendly", "witty" |
| Additional Context | Extra brand guidelines or notes |
| Default | Set as the default voice for all generations |

## How It Works

When a brand voice is selected during generation, its context is prepended to the AI prompt:

```
Brand Context:
Company: Acme Corp
Industry: SaaS
Target Audience: Small business owners
Tone: professional, friendly, approachable
Additional: We avoid jargon and speak directly to the reader.
```

This context shapes the AI's output to match your brand's style.

## Default Voice

One brand voice can be marked as the default. When set, it's automatically selected in the generation form and workflow settings.

## Limits

Brand voices are available on all plans. There is no limit on the number of brand voices per account.
